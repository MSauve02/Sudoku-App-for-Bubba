# Sudoku - Puzzle & Puns 🦔

A cute sudoku game with hedgehogs and dad jokes! Features 4×4, 6×6, and 9×9 grids with easy, medium, and hard difficulties.

## Quick Start (Local)

```bash
npm install
npm start
```

Opens at http://localhost:3000

## Deploy to Vercel (Easiest - Free)

1. Push this folder to a GitHub repo
2. Go to https://vercel.com and sign in with GitHub
3. Click "New Project" → import your repo
4. Click "Deploy" — done! You'll get a live URL to share

## Deploy to Netlify (Also Free)

1. Push this folder to a GitHub repo
2. Go to https://netlify.com and sign in with GitHub
3. Click "Add new site" → "Import an existing project"
4. Select your repo → click "Deploy"

## Deploy Manually (Any Static Host)

```bash
npm install
npm run build
```

Upload the `build/` folder to any static hosting (GitHub Pages, Cloudflare Pages, AWS S3, etc.)

## Features

- 🎯 Three grid sizes: 4×4, 6×6, 9×9
- 🏋️ Three difficulties: Easy, Medium, Hard
- 📝 Auto-candidate notes
- 💡 Hints system
- 😂 Dad joke reward on completion
- 🦔 Cute hedgehog theme
- ↻ Refresh for new puzzles
- ⌨️ Full keyboard support
